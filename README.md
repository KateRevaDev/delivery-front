This is the front-end part of the delivery project.
For the properly work of the roject you need to setup back-end part as well.

<!-- FOR UBUNTU -->

You need to do next steps for starting front-end:
1. Clone project
2. Install packages by command "npm install" in the terminal in project folder
3. Run the project by command "npm run dev" in the terminal in project folder
4. Enjoy the project on address "http://localhost:5173/"
